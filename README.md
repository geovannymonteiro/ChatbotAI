
# ChatbotAI


🤖 ChatBot AI com OpenAI + Streamlit Cloud

Este projeto consiste no desenvolvimento de um ChatBot inteligente utilizando a API da OpenAI, com interface web criada em Streamlit e hospedagem na Streamlit Cloud, permitindo que qualquer usuário interaja com o modelo de Inteligência Artificial diretamente pelo navegador.

O objetivo do projeto é demonstrar a integração entre aplicações Python, APIs de IA e deploy em nuvem, criando uma aplicação simples, funcional e acessível.

🚀 Funcionalidades

Interface web interativa para conversar com a IA

Integração com a API da OpenAI

Respostas geradas em tempo real

Interface simples e intuitiva usando Streamlit

Deploy na Streamlit Cloud para acesso público

Código aberto hospedado no GitHub

🛠️ Tecnologias Utilizadas

Python

Streamlit

OpenAI API

GitHub

Streamlit Cloud
